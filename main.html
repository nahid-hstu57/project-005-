Question-1: What are the different ways to select an element in the DOM?

You can select elements in the DOM using methods like getElementById("id"), getElementsByClassName("class"), getElementsByTagName("tag"), querySelector("selector"), and querySelectorAll("selector"). These allow you to access and manipulate specific elements in your webpage.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets the HTML content inside an element, including tags. innerText: Gets or sets only the visible text inside an element, ignoring hidden elements and extra spaces. textContent: Gets or sets all text inside an element, including hidden elements, but without parsing HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element listens for events on its child elements using event bubbling. Instead of adding event listeners to multiple child elements, a single listener is attached to the parent to handle events efficiently.

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM mechanism where an event starts from the target element and propagates up through its ancestors in the hierarchy. It allows event handlers on parent elements to respond to events triggered on their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element: Use document.createElement("tag"). Add an element: Use appendChild(), prepend(), or insertBefore(). Remove an element: Use removeChild() or element.remove().
